# MoneySavvy AI – Gen Z Financial Tweet Library

## Brand Context

**Audience:** Gen Z (17–23), Europe-based  
**Stage:** First paycheck, first rent, first savings, first investing curiosity  
**Goal:** Education → Trust → Referral  
**Voice:** Wry. Observational. Slightly sardonic. Intelligent. Non-preachy. Non-corporate.  

No dad-energy.  
No finance-bro hype.  
No generic motivation.
